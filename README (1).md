#Keyboard Event Logger 

An educational Python project built with Tkinter and `pynput` that demonstrates how keyboard events can be captured, displayed in a GUI, and saved into a log file.

⚠️ **Disclaimer**:  
I have created this project for **strictly for educational purposes**. It should only be used in a controlled environment to understand how keyboard input monitoring works. 
**Do not use this tool for malicious purposes.**

---

##Features
- Start and stop keyboard event logging  
- Real-time display of pressed keys in a Tkinter GUI  
- Save logs to a `.txt` file  
- Clear logs from the GUI
